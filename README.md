# Tandem Index

## Development

### Prerequisites

- Node.js (v20)
- Yarn

### Getting Started

1. Clone the repository.
2. Run `yarn` to install dependencies.
3. Run `yarn dev` to start the development server on [http://localhost:3000](http://localhost:3000).
